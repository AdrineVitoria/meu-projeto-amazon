# 🚀 Automação de Testes - Amazon Brasil

Este projeto contém a automação de testes funcionais para o site da Amazon Brasil, utilizando o **Robot Framework** com a biblioteca **SeleniumLibrary**.

## 🚀 O que este teste faz?
- Acessa o site da Amazon Brasil
- Realiza a busca por produtos específicos (ex: Iphone 15, PlayStation 5).
- Valida se os resultados da busca estão corretos através de verificações de texto e elementos na página.

## 💻 Tecnologias Utilizadas
* **Linguagem:** [Python](https://www.python.org/)
* **Framework:** [Robot Framework](https://robotframework.org/)
* **Biblioteca de Web:** [SeleniumLibrary](https://robotframework.org/SeleniumLibrary/)
* **IDE:** VS Code
